# Voice_Emotion_Recognition
The Voice Emotion Recognition System is a web-based application that identifies human emotions from speech input. 
Users can record their voice using a web interface, 
and the system analyzes the audio signal to predict emotions such as Happy, Sad, Angry, and Neutral.
